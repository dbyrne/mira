How to enable NINA's Advanced API
  1. In NINA: Plugins → Available → search "Advanced API" → install.
  2. Restart NINA.
  3. Settings → Advanced API → enable, default port 1888.
  4. Make sure NINA is running and has equipment connected.

If the dashboard says "unreachable" but NINA is running, check the port and that no firewall is blocking localhost.
